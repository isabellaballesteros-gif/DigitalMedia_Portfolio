Build an Interactive Web Narrative Project

About the Project

CLICK HERE to view the project. This project was the first project we did in class. We were tasked with experimenting with coding and using Cursor AI to create an interactive website with creative freedom. I chose to make an illusion of a choose your own adventure story with the viewer at a dinner, indecisive about what to order. The choices are found to be both bad endings which leads to an explanation about how in modern day society, we judge others and ourselves based on one meal. I resonate with this as someone who loves to eat and is not as thin as society sees as perfect. The project was a great way to start the class and I learned a lot about coding and Cursor AI. The website is very simple in design, but it was a good a starting point and a piece to compare to my later projects.
